# Algorytmy

*Implementacje podstawowych algorytmów w C++ i Pythonie.*

![C++](https://img.shields.io/badge/C%2B%2B-17-00599C?logo=cplusplus&logoColor=white)
![Python](https://img.shields.io/badge/Python-3.7-3776AB?logo=python&logoColor=white)

## 📖 Opis

Baza podstawowych algorytmów tworzonych w ramach nauki algorytmiki (technikum / samodzielna nauka) — klasyczne zadania z zakresu matematyki dyskretnej, obsługi plików i programowania proceduralnego.

## 📂 Struktura

| Folder | Zawartość |
|---|---|
| `podstawowe-algorytmy/` | Klasyczne algorytmy: silnia, NWD, NWW, liczby pierwsze, palindromy, ciągi, rozkład na czynniki pierwsze |
| `Algorytmika_AI/` | Zadania z algorytmiki i podstaw AI: hetmany (backtracking), kółko i krzyżyk, pseudokod |
| `odczyt-zapis-plikow/` | Obsługa plików (odczyt/zapis) w C++ i Pythonie |
| `pprog/` | Programowanie proceduralne w C++ |
| `pomoc/` | Pomocnicze skrypty testowe |

## 🛠️ Technologie

| Technologia | Szczegóły |
|---|---|
| C++17 | kompilator MinGW-w64 g++, konfiguracja w `.vscode/` |
| Python | 3.7 (interpreter wskazany w `.vscode/settings.json`) |
