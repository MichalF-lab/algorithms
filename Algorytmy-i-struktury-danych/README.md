# Algorytmy i struktury danych

*Listy zadań z przedmiotu Algorytmy i struktury danych (studia, PWr).*

![Python](https://img.shields.io/badge/Python-3-3776AB?logo=python&logoColor=white)

## 📖 Opis

Implementacje w Pythonie do kolejnych list zadań z przedmiotu Algorytmy i struktury danych — sortowanie, struktury kopcowe, programowanie dynamiczne.

## 📂 Struktura

| Folder / Plik | Zawartość |
|---|---|
| `lista1/` | Porównanie podstawowych algorytmów sortowania (przez wstawianie, bąbelkowe, 2 warianty merge sort) z licznikiem porównań/przestawień |
| `lista3/` | Quicksort (w tym wariant 3-drożny) i heapsort (binarny i ternarny) |
| `lista4/` | Programowanie dynamiczne: cięcie prętów (rod cutting) i najdłuższy wspólny podciąg (LCS) |
| `lista5/` | Wydawanie reszty (zachłannie i przez DP) oraz wybór zajęć (activity selection) |
| `lista6/` | Rekurencyjny bucket sort |
| `sorting_algorithms_analysis_plus.pdf` | Analiza porównawcza algorytmów sortowania |
| `OCENY.md` | Uwagi i oceny prowadzącego do poszczególnych list |
